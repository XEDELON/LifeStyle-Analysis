# Lifestyle and Well-being Data Analysis

## Project Overview
This project analyzes the "Lifestyle" dataset to uncover factors influencing work-life balance and overall well-being. Key steps include:

- Descriptive analysis and visualization.
- Correlation analysis.
- Decision Tree Classifier to predict work-life balance satisfaction.
- Linear Regression to predict overall well-being.
- Clustering analysis to group lifestyle patterns.

## Tools & Libraries:
- Python, Jupyter
- pandas, numpy
- seaborn, matplotlib
- scikit-learn

## Results:
- Key lifestyle factors affecting work-life balance were identified and predictions were made.
- Predictive models achieved [MSE: 17/R2: 0.99].
